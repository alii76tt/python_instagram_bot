# Requirement
colorama==0.4.4
selenium==3.141.0

## Properties
* Sign In
* Get Followers
* Get Following
* Follow User (can do more than one)
* Unfollow User (can do more than one)
* Who Don't Follow Me
* Watch Story

# Installation

```sh
$ git clone https://github.com/alii76tt/python_instagram_bot
$ cd python_instagram_bot
$ pip install -r requirements.txt
$ python instagram.py
```
